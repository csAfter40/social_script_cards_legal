## GitHub Pages

This repository is intended to be served as a GitHub Pages site.

1. Push changes to the `main` branch.
2. In the repository on GitHub, open **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Choose branch `main` and folder `/ (root)`, then save.

After deployment, the site is available at:

**https://csafter40.github.io/social_script_cards_legal/**

Use these URLs in the App Store, Google Play, and in-app links:

- Privacy Policy: `https://csafter40.github.io/social_script_cards_legal/privacy_policy.html`
- Terms & Conditions: `https://csafter40.github.io/social_script_cards_legal/terms_and_conditions.html`

## Updating

Edit the HTML files directly, then commit and push. When you change the legal text, update the **Last updated** date in the relevant document.
